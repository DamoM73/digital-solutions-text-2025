Figma is a web-based design tool used for creating user interfaces, prototypes, and collaborative design projects. It allows designers to create interactive prototypes that simulate the user experience of a website or application.

:::{seealso} Figma Activities
:class: dropdown
**Activity 1**
Use Figma to create a prototype of your computer's calculator application. Just do the standard calculator, don't worry about scientific functions.

**Activity 2**
Use Figma to create a prototype of a simple to-do list application. The prototype should include the following features:
- Add a new task
- Mark a task as completed
- Delete a task
- View all tasks
- Filter tasks by completed and pending

**Activity 3**
Use Figma to create a prototype of a simple weather application. The prototype should include the following features:
- Display current weather conditions
- Show a 5-day weather forecast
- Allow users to search for weather by city
- Display weather icons for different conditions
:::
